# F-NUR-007 — ตารางสรุปผู้ป่วยประกันสังคมที่ทำการผ่าตัดและตรวจฉุกเฉิน

เว็บแอปสำหรับพยาบาล **โรงพยาบาลวิภาราม อมตะนคร** ใช้แปลงข้อความขออนุมัติที่ได้รับทาง LINE
ให้กลายเป็นแถวในฟอร์ม F-NUR-007 Rev.01 แล้วพิมพ์เป็นฟอร์มราชการตอนสิ้นวัน

## ใช้งานอย่างไร

1. วางข้อความจาก LINE → กด **✨ วิเคราะห์ข้อความ**
   - วาง **1 ราย** → ระบบแยกลงช่องให้ตรวจก่อนบันทึก (ช่องที่ยังว่างจะขึ้นสีเหลืองเตือน)
   - วาง **หลายราย** (มีเลข 1. 2. 3. หรือหลาย HN) → ระบบแยกเป็นหลายแถวรวมเข้าตารางให้เลย
2. ตรวจ/แก้ข้อมูลที่ระบบแยกมาให้ → กด **➕ เพิ่มลงตาราง** (ถ้า HN ซ้ำกับที่มีในวันนั้น ระบบจะเตือน)
3. ทำซ้ำไปเรื่อย ๆ ทั้งวัน
4. สิ้นวันกด **🖨️ พิมพ์ / บันทึก PDF** (ฟอร์ม A4 แนวนอน) หรือ **⬇️ Excel (CSV)**
5. อยากส่งสรุปให้ทีมเร็ว ๆ กด **📄 ส่งรายงาน PDF เข้า LINE** หรือ **🖼️ ส่งรายงานรูปภาพ (PNG)**
   → ระบบสร้างรายงาน A4 แนวนอน (1 รายการก็เต็มหน้า, ล้นก็ขึ้นหลายหน้าโดยมีหัวฟอร์ม/โลโก้/หัวตารางซ้ำทุกหน้า) แล้วเปิดหน้าต่างแชร์ไป LINE

### แยกโรค / การตรวจให้แม่นขึ้น

ระบบแยก "ผลวินิจฉัย (โรค)" กับ "การผ่าตัด/ตรวจพิเศษ" อัตโนมัติจากคำสำคัญ (แล็บ/ภาพถ่าย = การตรวจ, ชื่อโรค = วินิจฉัย)
ถ้ามีคำที่ระบบยังแยกไม่ได้ เพิ่มเองได้ที่ **ตั้งค่าฟอร์ม → คำที่ให้ถือเป็น...** ระบบจะจดจำและใช้แยกให้ครั้งต่อไป
รองรับคำนำหน้าชื่อ ไทย (นาย/นาง/นางสาว/ด.ช./ด.ญ.) และอังกฤษ (Mr./Mrs./Ms./Miss) — ถ้าไม่มีคำนำหน้าจะเติม "คุณ" ให้

## การส่งต่อเวร

แอปเก็บข้อมูลไว้ในเครื่องแต่ละคน (localStorage) ไม่ได้ซิงก์กันเอง

- คนที่บันทึกอยู่กด **📤 ส่งต่อเวร** → เลือกส่ง **ข้อความ** ไปแชท LINE ของเวรถัดไป (ใช้ได้ทั้ง Android และ iPad ไม่ต้องเปิดไฟล์)
- คนรับเวร **กดค้างที่ข้อความใน LINE เพื่อคัดลอก** นำมาวางในช่อง **“วางข้อความจาก LINE”** แล้วกด **✨ วิเคราะห์ข้อความ**
- ระบบรวมรายการให้โดยข้ามรายการที่ซ้ำ (เทียบจาก HN + ชื่อ + ผลวินิจฉัย) เลขลำดับจึงต่อเนื่องเสมอ

## ติดตั้งบน iPad

เปิดลิงก์ของเว็บนี้ใน **Safari** → ปุ่มแชร์ → **เพิ่มไปยังหน้าจอโฮม**

จะได้ไอคอนแอปบนหน้าจอ เปิดใช้งานได้แม้ไม่มีอินเทอร์เน็ต (ผ่าน service worker)
ฟอนต์ Sarabun ฝังมาในไฟล์แล้ว ไม่ต้องโหลดจากภายนอก

## ข้อควรระวัง

ข้อมูลอยู่ในเบราว์เซอร์ของแต่ละเครื่อง — ถ้ากด "ล้างประวัติและข้อมูลเว็บไซต์" ใน Safari
ข้อมูลของวันนั้นจะหายไปด้วย ควรกด **ส่งต่อเวร (ไฟล์)** เก็บสำเนาไว้ก่อนสิ้นวันทุกครั้ง

**ห้ามนำรูปถ่ายฟอร์มที่กรอกแล้ว หรือไฟล์ JSON ที่มีข้อมูลผู้ป่วย มาใส่ใน repo นี้**
repo นี้เป็นสาธารณะ และตัวแอปเองไม่มีข้อมูลผู้ป่วยอยู่ภายใน

## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|---|---|
| `index.html` | ตัวแอปทั้งหมด (HTML + CSS + JS + ฟอนต์ Sarabun ฝังในไฟล์) |
| `sw.js` | service worker สำหรับใช้งานออฟไลน์ |
| `manifest.webmanifest` | ข้อมูลแอปสำหรับ "เพิ่มไปยังหน้าจอโฮม" |
| `icon-*.png` | ไอคอนแอป |

แก้ `index.html` เมื่อไหร่ ให้เปลี่ยนเลขเวอร์ชันใน `sw.js` (`CACHE = 'fnur007-v1'`) ด้วย
ไม่อย่างนั้นเครื่องที่ติดตั้งไปแล้วจะยังใช้เวอร์ชันเก่าที่แคชไว้
